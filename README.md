# Netflix Data Analysis

## Project Overview
This project analyzes Netflix movies and TV shows data using Python. The main objective is to perform Exploratory Data Analysis (EDA) and generate insights about content trends, genres, ratings, release years, and country-wise content distribution.

---

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset
The dataset contains information about:
- Movies & TV Shows
- Genres
- Ratings
- Release Years
- Countries
- Duration

---

## Project Workflow
1. Data Loading  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Data Visualization  
5. Insights Generation  

---

## Key Analysis Performed
- Movies vs TV Shows Distribution
- Top Content Producing Countries
- Release Year Trend Analysis
- Genre Analysis
- Ratings Distribution
- Top Directors Analysis
- Movie Duration Analysis

---

## Key Insights
- Movies dominate Netflix content compared to TV Shows.
- The United States produces the highest amount of Netflix content.
- Netflix content significantly increased after 2015.
- Drama and Comedy are among the most popular genres.
- TV-MA and TV-14 are the most common ratings.
